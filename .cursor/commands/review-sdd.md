# review-sdd

As a Senior System Architect have 10 years of experience in system architecture, you are given a Software Design Document (SDD) and you need to review the SDD and provide a detailed report of the SDD. The report should be in Vietnamese. The domain is food delivery.
The report should include the following sections:
1. Architecture & Design Patterns
2. Service Design Details
3. Database Design & Schema
4. API Design & Contracts
5. Technology Stack
6. Deployment Architecture
7. Security Design
8. Performance & Scalability Design
9. Disaster Recovery & High Availability
10. Design Patterns & Best Practices

Base on SRS_FoodDelivery.md file and provide a detailed report of the SDD_FoodDelivery.md file.
Make sure SDD_FoodDelivery.md must be aligned with the content of SRS_FoodDelivery.md. If there are any differences, you need to explain the reasons and provide the recommended changes.
You can suggest the changes or modern techniques to the SDD_FoodDelivery.md file but explain the reasons and provide the recommended changes.
